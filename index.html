LOL-Mentor — Project Overview

1. Introduction

I am a Chinese League of Legends content creator and streamer developing a personal analysis tool for educational purposes.
This project focuses on studying high-ranking and professional players’ gameplay on the Korean (KR) server, which is known for its competitive environment and strategic depth.

  • Personal learning and improvement
  • Creating educational content for my small private viewer community
  • Understanding gameplay trends, champion picks, and itemization patterns

2. Purpose of the Project

The primary objective of this tool is to analyze gameplay patterns of skilled KR players and present the insights in a clear and instructional format.

  • Identifying effective macro/micro strategies
  • Understanding current meta trends
  • Studying decision-making patterns through match data
  • Supporting educational commentary in my content

This project is not commercial, does not provide competitive advantages, and will never be used for public player profiling or ranking.

3. Target Audience

  • Myself (developer/creator)
  • A small private community of viewers who follow my educational content

No public access, public API exposure, or downloadable tools are provided.

4. Features

Player Lookup

  • Search KR summoner name
  • Retrieve PUUID and ranked stats

Match History Analysis

  • Fetch recent match IDs from Match API
  • Analyze game durations, roles, champions, items
  • Identify gameplay patterns and strategic decisions

Educational Insights

  • Highlight strong plays, item choices, and meta adaptations
  • Present information positively and constructively
  • Generate insights for educational videos or stream discussions

5. APIs Used

  • Match API (Match-V5) — retrieving match IDs and detailed game information
  • Summoner API (Summoner-V4) — obtaining player identifiers such as PUUID
  • League API (League-V4) — for ranked tier, LP, and competitive context

These APIs are used only for data analysis and educational purposes.

6. Data Usage & Privacy

  • No player personal data is stored permanently
  • No user accounts or login systems are used
  • No data is shared with third parties
  • All data is processed locally
  • No automated monitoring or large-scale data harvesting
  • No competitive advantages or exploitative mechanics
  • Only high-ranking and pro matches are analyzed for educational content

7. Distribution

  • Not distributed publicly
  • Not monetized
  • Not commercial
  • Used solely for personal educational content creation

8. Project Status

Currently under active development. Planned stages:

  • Match analysis UI
